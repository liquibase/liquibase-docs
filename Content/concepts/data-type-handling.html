Data Type Handling

When you specify a data type in the type field of a or tag, such as the <column type="XYZ"> field, evaluates the type to send it to your database.

implements many generic types natively. When necessary, converts the data type you specify into a different type preferred by your database.

For example, if you specify the data type BOOLEAN in your , passes it to a Snowflake database as a BOOLEAN. However, on a PostgreSQL database, converts it to a BIT.
