dropView

Drops an existing view.

Available s

Name Description Required for Supports Since
catalogName all 3.0
ifExists Appends IF EXISTS to the DROP VIEW statement. If ifExists=true, the view is only dropped if it already exists, but the migration continues even if the view does not exist. If ifExists=false and the view does not exist, the database returns an error. Default: false.   all except oracle, firebird 4.19.0
schemaName all
viewName all all
XML example 
<changeSet  author="liquibase-docs"  id="dropView-example">  
    <dropView  catalogName="cat"  
            schemaName="public"  
            viewName="v_person"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  dropView-example  
    author:  liquibase-docs  
    changes:  
    -  dropView:  
        catalogName:  cat  
        schemaName:  public  
        viewName:  v_person
 JSON example

}
 SQL example 
DROP  VIEW  cat.v_person;
