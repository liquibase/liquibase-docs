dropDefaultValue

Removes the database default value for a column.

Available s

Name Description Required for Supports Since
catalogName all 3.0
columnDataType informix all
columnName to drop the default value from all all
schemaName all
tableName containing the column to drop the default value from all all
XML example 
<changeSet  author="liquibase-docs"  id="dropDefaultValue-example">  
    <dropDefaultValue  catalogName="cat"  
            columnDataType="int"  
            columnName="id"  
            schemaName="public"  
            tableName="person"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  dropDefaultValue-example  
    author:  liquibase-docs  
    changes:  
    -  dropDefaultValue:  
        catalogName:  cat  
        columnDataType:  int  
        columnName:  id  
        schemaName:  public  
        tableName:  person
 JSON example 
{
  "changeSet":  {
    "id":  "dropDefaultValue-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "dropDefaultValue":  {
        "catalogName":  "cat",
        "columnDataType":  "int",
        "columnName":  "id",
        "schemaName":  "public",
        "tableName":  "person"
      }
    }]
  }
}
 SQL example 
ALTER  TABLE  cat.person  ALTER  id  DROP  DEFAULT;
