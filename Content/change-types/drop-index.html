dropIndex

The dropIndex drops an existing index on a column.

Uses

You can typically use the dropIndex when you want to remove the specific index that identifies records in your database.

Once an index has been dropped, all metadata is removed. The index canâ€™t be rebuilt and you must create it again. Also, if you drop a clustered index, the table will lose its identifiers.

Running the dropIndex

To drop an existing index in your database, follow these steps:

  1. Add the dropIndex to your with the needed s as it is shown in the examples.
  2. Deploy your by running the update command:
    3. liquibase update

<changeSet author="liquibase-docs" id="dropIndex-example">
    <dropIndex indexName="idx_address"
            schemaName="public"
            tableName="person"/>  
</changeSet>
