addCheckConstraint

Adds a check constraint to an existing column or set of columns in a table.

Run addCheckConstraint

Available s

Name Description Required for Supports Since
catalogName all 3.0
constraintBody all all
constraintName all all
disabled all
schemaName all
tableName

to create the check constraint on

 all all
validate all
