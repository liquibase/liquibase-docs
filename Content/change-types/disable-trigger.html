disableTrigger

Disables a trigger.

Available s

Name Description Required for Supports Since
catalogName oracle, postgresql
schemaName all
scope all
tableName all
triggerName all all
XML example 
<changeSet  author="liquibase-docs"  id="disableTrigger-example">  
    <pro:disableTrigger  catalogName="cat"  
            schemaName="public"  
            scope="A String"  
            tableName="person"  
            triggerName="A String"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  disableTrigger-example  
    author:  liquibase-docs  
    changes:  
    -  disableTrigger:  
        catalogName:  cat  
        schemaName:  public  
        scope:  A String  
        tableName:  person  
        triggerName:  A String
 JSON example 
{
  "changeSet":  {
    "id":  "disableTrigger-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "disableTrigger":  {
        "catalogName":  "cat",
        "schemaName":  "public",
        "scope":  "A String",
        "tableName":  "person",
        "triggerName":  "A String"
      }
    }]
  }
}
 SQL example 
DISABLE  TRIGGER  [public].[A  String]    ON  cat.[public].person;
