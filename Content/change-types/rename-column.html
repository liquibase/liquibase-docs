renameColumn

Renames an existing column.

Available s

Name Description Required for Supports Since
catalogName all 3.0
columnDataType mariadb, mysql all
newColumnName New name for the column all all
oldColumnName Existing name of the column to rename all all
remarks all
schemaName all
tableName

containing the column to rename

 all all
XML example 
<changeSet  author="liquibase-docs"  id="renameColumn-example">  
    <renameColumn  catalogName="cat"  
            columnDataType="int"  
            newColumnName="full_name"  
            oldColumnName="name"  
            remarks="A String"  
            schemaName="public"  
            tableName="person"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  renameColumn-example  
    author:  liquibase-docs  
    changes:  
    -  renameColumn:  
        catalogName:  cat  
        columnDataType:  int  
        newColumnName:  full_name  
        oldColumnName:  name  
        remarks:  A String  
        schemaName:  public  
        tableName:  person
 JSON example

  "changeSet":  {

 SQL example 
ALTER  TABLE  cat.person  CHANGE  name  full_name  INT  COMMENT  'A String';
