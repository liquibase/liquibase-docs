markUnused

The markUnused is an Oracle-specific that marks a column as unused. The markUnused helps to avoid database slowdowns caused by dropping a populated column.

Uses

You can typically use the markUnused to mark columns as unused rather than dropping them immediately, which allows you to decide the best time to drop those columns.

Available s

Name Description Required for Supports
catalogName all
columnName all all
schemaName all
tableName all all
XML example 
<changeSet  author="liquibase-docs"  id="markUnused-example">  
    <pro:markUnused  catalogName="cat"  
            columnName="id"  
            schemaName="public"  
            tableName="person"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  markUnused-example  
    author:  liquibase-docs  
    changes:  
    -  markUnused:  
        catalogName:  cat  
        columnName:  id  
        schemaName:  public  
        tableName:  person
 JSON example 
{
  "changeSet":  {

    "changes":  [{
      "markUnused":  {
        "catalogName":  "cat",

  }
}
 SQL example 
ALTER  TABLE  cat.person  SET  UNUSED  COLUMN  id;
