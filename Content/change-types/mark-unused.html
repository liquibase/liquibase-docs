markUnused

The markUnused  is an Oracle-specific that marks a column as unused. The markUnused  helps to avoid database slowdowns caused by dropping a populated column.

Uses

You can typically use the markUnused  to mark columns as unused rather than dropping them immediately, which allows you to decide the best time to drop those columns.

Run markUnused
