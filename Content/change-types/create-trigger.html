createTrigger

The createTrigger creates the trigger in a database.

Uses

You can typically use the createTrigger  when you want to add a trigger to the table which will be automatically activated when a specific event occurs in your database. It can include catalog, schema, and table name s listed in this documentation.

Running the createTrigger 

To create a trigger in your database, follow these steps:

  1. Add the createTrigger to your with the needed s as it is shown in the examples.
  2. Deploy your by running the update command:
    3. liquibase update

Now, you should see that the trigger you've specified is added.
