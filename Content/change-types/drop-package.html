dropPackage

Drops an existing package.

Available s

Name Description Required for Supports Since
catalogName all
packageName all all
schemaName all
XML example 
<changeSet  author="liquibase-docs"  id="dropPackage-example">  
    <pro:dropPackage  catalogName="cat"  
            packageName="A String"  
            schemaName="public"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  dropPackage-example  
    author:  liquibase-docs  
    changes:  
    -  dropPackage:  
        catalogName:  cat  
        packageName:  A String  
        schemaName:  public
 JSON example 
{
  "changeSet":  {
    "id":  "dropPackage-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "dropPackage":  {
        "catalogName":  "cat",
        "packageName":  "A String",
        "schemaName":  "public"
      }
    }]
  }
}
 SQL example 
DROP  PACKAGE  cat.`A String`;
