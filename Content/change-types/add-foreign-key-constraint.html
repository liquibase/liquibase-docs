addForeignKeyConstraint

Adds a foreign key constraint to an existing column.

Available s

Name Description Required for Supports
baseColumnNames (s) to place the foreign key constraint on. Comma-separate if multiple all all
baseTableCatalogName of the base table all
baseTableName containing the column to constraint all all
baseTableSchemaName of the base table all
constraintName all all
deferrable oracle, postgresql, sqlite
initiallyDeferred oracle, postgresql, sqlite
onDelete asany, db2, db2z, derby, firebird, h2, hsqldb, informix, ingres, mariadb, mssql, mysql, oracle, postgresql
onUpdate all except db2z
referencedColumnNames (s) the foreign key points to. Comma-separate if multiple all all
referencedTableCatalogName of the referenced table all
referencedTableName the foreign key points to all all
referencedTableSchemaName of the referenced table all
validate all
referencesUniqueColumn all
XML example 
<changeSet  author="liquibase-docs"  id="addForeignKeyConstraint-example">  
    <addForeignKeyConstraint  baseColumnNames="person_id"  
            baseTableCatalogName="cat"  
            baseTableName="address"  
            baseTableSchemaName="public"  
            constraintName="fk_address_person"  
            deferrable="true"  
            initiallyDeferred="true"  
            onDelete="CASCADE"  
            onUpdate="RESTRICT"  
            referencedColumnNames="id"  
            referencedTableCatalogName="cat"  
            referencedTableName="person"  
            referencedTableSchemaName="public"  
            validate="true"/>  
</changeSet>
