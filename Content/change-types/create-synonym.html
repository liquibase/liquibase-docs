createSynonym

Creates a synonym.

Available s

Name Description Required for Supports Since
objectCatalogName of the object the synonym represents all
objectName all all
objectSchemaName of the object the synonym represents all
objectType db2 all
private db2z all
replaceIfExists

Boolean. If the synonym defined by synonymName already exits, alter it instead of creating it. Default: false.

 all
synonymCatalogName of the synonym all
synonymName all all
synonymSchemaName of the synonym all
XML example 
<changeSet  author="liquibase-docs"  id="createSynonym-example">  
    <pro:createSynonym  objectName="A String"  
            objectType="A String"  
            private="true"  
            replaceIfExists="false"  
            synonymName="A String"/>  
</changeSet>
 YAML example
