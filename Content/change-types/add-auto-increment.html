addAutoIncrement

Converts an existing column to be an auto-increment (a.k.a 'identity') column.

Run addAutoIncrement

Available s

Description Required for Supports
catalogName -- all
columnDataType asany, h2, hsqldb, informix, ingres, mariadb, mysql, sybase, unsupported all
columnName all all
defaultOnNull -- oracle
generationType -- all
incrementBy -- all
schemaName -- all
startWith -- all
tableName all all
