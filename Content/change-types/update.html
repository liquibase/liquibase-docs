update

The update updates the data in an existing table.

Uses

You can typically use the update when you want to update a table in your file and then deploy this update to your database. It can include columns and another values listed in this documentation.

Running the update

To update a table for your database, follow these steps:

  1. Add the update to your with the needed s as it is shown in the examples.
  2. Deploy your by running the update command:
    3. liquibase update

Now, you should see an updated table.

You can use the update in combination with loadData and loadUpdateData s.

XML example 
<changeSet author="liquibase-docs" id="update-example">
    <update catalogName="liquibase" schemaName="public" tableName="person">
        <column name="address" value="address value"/>
        <where>name='Bob'</where>
    </update>
</changeSet>
 YAML example 
-  changeSet:
    id:  update-example
    author:  liquibase-docs
    changes:
    -  update:
        catalogName:  liquibase
        columns:
        -  column:
            name:  address
            value:  address value
        schemaName:  public
        tableName:  person
        where:  name='Bob'
 JSON example 
{
  "changeSet": {
    "id": "update-example",
    "author": "liquibase-docs",
    "changes": [
      {

            }

      }

  }
}
 SQL example 
UPDATE  liquibase.person  SET  address  =  'address value'  WHERE  name='Bob';
