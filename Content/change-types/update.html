update

The update updates the data in an existing table.

Uses

You can typically use the update when you want to update a table in your file and then deploy this update to your database. It can include columns and another values listed in this documentation.

Running the update

Now, you should see an updated table.

You can use the update in combination with loadData and loadUpdateData s.
