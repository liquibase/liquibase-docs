delete

Deletes data from an existing table.

Available s

Name Description Required for Supports Since
catalogName all 3.0
schemaName all
tableName all all

Nested tags

Name Description Required for Supports Multiple allowed
where all no
whereParams all no
XML example 
<changeSet  author="liquibase-docs"  id="delete-example">  
    <delete  catalogName="cat"  
            schemaName="public"  
            tableName="person">  
        <where>name='Bob'</where>  
    </delete>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  delete-example  
    author:  liquibase-docs  
    changes:  
    -  delete:  
        catalogName:  cat  
        schemaName:  public  
        tableName:  person  
        where:  name='Bob'
 JSON example 
{
  "changeSet": {
      "id":  "delete-example",
      "author":  "liquibase-docs",
      "changes":  [{
        "delete":  {
          "catalogName":  "cat",
          "schemaName":  "public",
          "tableName":  "person",
          "where":  "name='Bob' and :name=:value or id=:value",
          "whereParams":  [{
            "param":  {
              "name":  "id",
              "value":  "str"
            }
          },{
            "param":  {
              "valueNumeric":  123
            }
          }]
        }
      }]
  }
}
 SQL example 
DELETE  FROM  cat.person  WHERE  name='Bob';
