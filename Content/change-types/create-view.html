createView

Creates a new database view.

Run createView

Available s

Name Description Required for Supports
catalogName all
encoding all
fullDefinition all
path Path to file containing view definition. Using the path is an alternative to selectQuery. all
relativeToChangelogFile all
remarks all
replaceIfExists

Use CREATE OR REPLACE syntax

Available in 1.5 and later

To apply the selectQuery value in XML, set it as a body of the tag: 



    <changeSet  author="liquibase-docs"  id="createView-example">
        <createView  catalogName="cat"
            encoding="UTF-8"
            fullDefinition="false"
            remarks="A String"
            replaceIfExists="false"
            schemaName="public"
            viewName="v_person">select id, name from person where id > 10</createView>
    </changeSet>

</databaseChangeLog>

Using the path is an alternative to selectQuery:



    <changeSet  author="liquibase-docs"  id="createView-example">
        <createView  catalogName="cat"
            encoding="UTF-8"
	        path="A String"
	        relativeToChangelogFile="true"
	        remarks="A String"
	        replaceIfExists="false"
	        schemaName="public"
	        viewName="v_person"></createView>
    </changeSet>

</databaseChangeLog>

Without parenthesis:

--liquibase formatted sql

--changeset liquibase-docs:createView-example
CREATE  VIEW  cat.v_person  AS  select  id,  
 name  from  person  where  id  >  10;

With parenthesis ( 4.25.1+):

--liquibase formatted sql

--changeset liquibase-docs:createView-example
CREATE  VIEW  cat.v_person  AS(select  id,  
 name  from  person  where  id  >  10;)
