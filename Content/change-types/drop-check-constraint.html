dropCheckConstraint

Drops an existing check constraint.

Run dropCheckConstraint

Available s

Name Description Required for Supports Since
catalogName all 3.0
constraintName all all
schemaName all
tableName to drop the check constraint from all all

    id:  dropCheckConstraint-example
    author:  liquibase-docs
    changes:

        catalogName:  cat

        schemaName:  public
        tableName:  person
