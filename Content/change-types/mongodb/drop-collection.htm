dropCollection

dropCollection is a in the and extensions for MongoDB that removes a collection or view from the database.

Uses

dropCollection is one such . You can use it to drop an existing collection from your database.

Run

Available s

For more information, see db.collection.drop().

Name Type Description Requirement
collectionName String to drop Required
