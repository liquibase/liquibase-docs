adminCommand

adminCommand is a in the and extensions for MongoDB that provides a helper to run specified database commands against the admin database.

Uses

adminCommand is one such . You can use it to run commands on the built-in admin database. You must be an administrator in MongoDB to successfully use this . For more information, see Manage Users and Roles.

If you want to run commands on another database, use runCommand instead.

Run

Available s

For more information, see db.adminCommand().

Name Type Description Requirement
command Raw JSON Required
