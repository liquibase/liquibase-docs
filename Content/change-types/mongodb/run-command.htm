runCommand

runCommand is a in the and extensions for MongoDB that provides a helper to run specified database commands.

Uses

runCommand is one such . You can use it to run commands on the current database. This is the preferred method to issue database commands as it provides a consistent interface between the shell and drivers.

If you want to run commands on the built-in admin database, use adminCommand instead.

For more information, see db.runCommand().
