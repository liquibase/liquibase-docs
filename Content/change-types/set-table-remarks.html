setTableRemarks

Sets remarks on a table.

Available s

Name Description Required for Supports Since
catalogName all 3.0
remarks all all
schemaName all
tableName all all
XML example 
<changeSet  author="liquibase-docs"  id="setTableRemarks-example">  
    <setTableRemarks  catalogName="cat"  
            remarks="A String"  
            schemaName="public"  
            tableName="person"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  setTableRemarks-example  
    author:  liquibase-docs  
    changes:  
    -  setTableRemarks:  
        catalogName:  cat  
        remarks:  A String  
        schemaName:  public  
        tableName:  person
 JSON example 
{
  "changeSet":  {
    "id":  "setTableRemarks-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "setTableRemarks":  {
        "catalogName":  "cat",
        "remarks":  "A String",
        "schemaName":  "public",
        "tableName":  "person"
      }
    }]
  }
}
 SQL example 
ALTER  TABLE  cat.person  COMMENT  =  'A String';
