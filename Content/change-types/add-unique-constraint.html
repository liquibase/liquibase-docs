addUniqueConstraint

Adds a unique constraint to an existing column or set of columns.

Available s

Name Description Required for Supports
catalogName all
clustered mssql
columnNames (s) to create the unique constraint on. Comma separated if multiple all all
constraintName all
deferrable all
disabled all
forIndexCatalogName of the index to associate with the constraint all
forIndexName to associate with the constraint oracle
forIndexSchemaName of the index to associate with the constraint all
initiallyDeferred all
schemaName all
tableName to create the unique constraint on all all
tablespace

for the index

 all
validate all
XML example 
<changeSet  author="liquibase-docs"  id="addUniqueConstraint-example">  
    <addUniqueConstraint  catalogName="cat"  
            clustered="true"  
            columnNames="id, name"  
            constraintName="const_name"  
            deferrable="true"  
            disabled="true"  
            forIndexName="A String"  
            initiallyDeferred="true"  
            schemaName="public"  
            tableName="person"  
            tablespace="A String"  
            validate="true"/>  
</changeSet>
