renameTrigger

Renames a trigger.

Available s

Name Description Required for Supports Since
catalogName Name of the catalog oracle, postgresql
newTriggerName all all
oldTriggerName all all
schemaName Name of the schema all
tableName Name of the table postgresql all
XML example 
<changeSet  author="liquibase-docs"  id="renameTrigger-example">  
    <pro:renameTrigger  catalogName="cat"  
            newTriggerName="A String"  
            oldTriggerName="A String"  
            schemaName="public"  
            tableName="person"/>  
</changeSet>
 YAML example 
changeSet:  
  id:  renameTrigger-example  
  author:  liquibase-docs  
  changes:  
  -  renameTrigger:  
      catalogName:  cat  
      newTriggerName:  A String  
      oldTriggerName:  A String  
      schemaName:  public  
      tableName:  person
 JSON example 
{
  "changeSet":  {
    "id":  "renameTrigger-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "renameTrigger":  {
        "catalogName":  "cat",
        "newTriggerName":  "A String",
        "oldTriggerName":  "A String",
        "schemaName":  "public",
        "tableName":  "person"
      }
    }]
  }
}
 SQL example 
sp_rename  '[public].A String',  
 'A String';
