dropFunction

Drops an existing function.

Available s

Name Description Required for Supports Since
catalogName asany, db2, db2z, derby, firebird, h2, hsqldb, informix, ingres, mariadb, mysql, oracle, postgresql, sqlite, sybase
functionName all all
schemaName all
XML example 
<changeSet  author="liquibase-docs"  id="dropFunction-example">  
    <pro:dropFunction  catalogName="cat"  
            functionName="A String"  
            schemaName="public"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  dropFunction-example  
    author:  liquibase-docs  
    changes:  
    -  dropFunction:  
        catalogName:  cat  
        functionName:  A String  
        schemaName:  public
 JSON example 
{
  "changeSet":  {

    "author":  "liquibase-docs",
    "changes":  [{
      "dropFunction":  {
        "catalogName":  "cat",
        "functionName":  "A String",

      }
    }]
  }
}
 SQL example 
DROP  FUNCTION  cat.`A String`;
