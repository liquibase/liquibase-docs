dropAllForeignKeyConstraints

Drops all foreign key constraints for a table.

Run dropAllForeignKeyConstraints

Available s

Name Description Required for Supports Since
baseTableCatalogName of the table containing columns constrained by foreign keys all 3.0
baseTableName containing columns constrained by foreign keys all all
baseTableSchemaName of the table containing columns constrained by foreign keys all
