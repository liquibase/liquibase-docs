renameTable

Renames an existing table.

Available s

Name Description Required for Supports
catalogName all
newTableName New name for the table all all
oldTableName Existing name of the table to rename all all
schemaName all
XML example 
<changeSet  author="liquibase-docs"  id="renameTable-example">  
    <renameTable  catalogName="cat"  
            newTableName="employee"  
            oldTableName="person"  
            schemaName="public"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  renameTable-example  
    author:  liquibase-docs  
    changes:  
    -  renameTable:  
        catalogName:  cat  
        newTableName:  employee  
        oldTableName:  person  
        schemaName:  public
 JSON example 
{
  "changeSet":  {
    "id":  "renameTable-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "renameTable":  {
        "catalogName":  "cat",
        "newTableName":  "employee",
        "oldTableName":  "person",
        "schemaName":  "public"
      }
    }]
  }
}
 SQL example 
ALTER  TABLE  cat.person  RENAME TO  cat.employee;
