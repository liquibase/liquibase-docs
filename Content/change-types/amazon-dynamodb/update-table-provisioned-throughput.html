updateTableProvisionedThroughput

updateTableProvisionedThroughput is a in the DynamoDB Pro extension that updates an existing table. It only works for tables that use the PROVISIONED billing mode.

Uses

You can use updateTableProvisionedThroughput to update tables that use the PROVISIONED billing mode.

If you try to use updateTableProvisionedThroughput to update a table that uses the PAY_PER_REQUEST billing mode, returns an error. This occurs even if you're trying to switch to the PROVISIONED billing mode. Instead, you must use updateDynamoTable.

Run updateTableProvisionedThroughput

Available s

For more information, see DynamoDB API UpdateTable.

Name Type Description Requirement
tableName String

to update

 Required
readCapacityUnits Integer

The maximum number of reads per second before DynamoDB returns a ThrottlingException.

 Optional
writeCapacityUnits Integer The maximum number of writes per second before DynamoDB returns a ThrottlingException. Optional
