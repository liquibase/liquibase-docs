deleteGlobalSecondaryIndex

deleteGlobalSecondaryIndex is a in the DynamoDB Pro extension that deletes a global secondary index from a table. For more information, see Using Global Secondary Indexes in DynamoDB.

Uses

Run deleteGlobalSecondaryIndex
