dropPackageBody

Drops an existing package body.

Available s

Name Description Required for Supports Since
catalogName all
packageBodyName all all
schemaName all
XML example 
<changeSet  author="liquibase-docs"  id="dropPackageBody-example">  
    <pro:dropPackageBody  catalogName="cat"  
            packageBodyName="A String"  
            schemaName="public"/>  
</changeSet>
 YAML example 
changeSet:  
  id:  dropPackageBody-example  
  author:  liquibase-docs  
  changes:  
  -  dropPackageBody:  
      catalogName:  cat  
      packageBodyName:  A String  
      schemaName:  public
 JSON example 
{
  "changeSet":  {
    "id":  "dropPackageBody-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "dropPackageBody":  {
        "catalogName":  "cat",
        "packageBodyName":  "A String",
        "schemaName":  "public"
      }
    }]
  }
}
 SQL example 
DROP  PACKAGE  BODY  cat.`A String`;
