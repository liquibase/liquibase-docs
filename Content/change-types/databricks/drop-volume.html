dropVolume

dropVolume is a in the Databricks extension that drops a volume. It is available in the Databricks extension 1.0.0 and later.

Uses

You can use this change to drop an existing Databricks volume from your database.

For more information, see Databricks SQL Reference: DROP VOLUME.

Run dropVolume

Available s

Name Type Description Requirement
volumeName String to drop Required
catalogName String Optional
ifExists Boolean Optional
schemaName String Optional
