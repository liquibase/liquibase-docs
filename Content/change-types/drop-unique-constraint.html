dropUniqueConstraint

Drops an existing unique constraint.

Available s

Name Description Required for Supports Since
catalogName all 3.0
constraintName all all
schemaName all
tableName to drop the unique constraint from all all
uniqueColumns sybase
XML example 
<changeSet  author="liquibase-docs"  id="dropUniqueConstraint-example">  
    <dropUniqueConstraint  catalogName="cat"  
            constraintName="const_name"  
            schemaName="public"  
            tableName="person"  
            uniqueColumns="name"/>  
</changeSet>
 YAML example 
-  changeSet:  
    id:  dropUniqueConstraint-example  
    author:  liquibase-docs  
    changes:  
    -  dropUniqueConstraint:  
        catalogName:  cat  
        constraintName:  const_name  
        schemaName:  public  
        tableName:  person  
        uniqueColumns:  name
 JSON example 
{
  "changeSet":  {
    "id":  "dropUniqueConstraint-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "dropUniqueConstraint":  {
        "catalogName":  "cat",
        "constraintName":  "const_name",
        "schemaName":  "public",
        "tableName":  "person",
        "uniqueColumns":  "name"
      }
    }]
  }
}
 SQL example 
ALTER  TABLE  cat.person  DROP  KEY  const_name;
