update-to-tag-sql

The update-to-tag-sql command is a helper command that inspects the SQL will run while using the update-to-tag command.

Uses

The update-to-tag-sql command is used to inspect the raw SQL before running the update-to-tag command, so you can correct any issues that may arise before running the command. uses the raw SQL to apply database changes you have added to the file based on the tag specified.

Syntax

To run the update-to-tag-sql command, specify the driver, classpath, and URL in the properties file. You can also specify them from the command line.

Then run the update-to-tag-sql command:

liquibase update-to-tag-sql --tag=myTag --changelog-file=example-changelog.xml

Parameters

  • CLI
  • Flow
  • liquibase.properties
  • JAVA_OPTS
  • Environment variable

* checks the and any nested s for the definitions of the s to update.
