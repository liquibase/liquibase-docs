clear-checksums

Syntax

To run the clear-checksums command, specify the database driver, classpath, and URL the properties file. For more information, see Create and Configure a [%=General.liquiPropFile%] File. You can also specify these properties in your command line.

Then run the clear-checksums command:

liquibase clear-checksums

Command arguments

Definition Requirement
--url Required
--default-catalog-name Optional
--default-schema-name Optional
--driver Optional
--driver-properties-file Optional
--password Optional
--username Optional

Output

When successful, the clear-checksums command produces the following output:

Liquibase Pro 3.9.0  licensed to Liquibase Pro Customer until Tue Sep 22 19:00:00 CDT 2020
Liquibase command 'clear-checksums' was executed successfully.
