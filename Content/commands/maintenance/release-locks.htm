release-locks

Syntax

To run the release-locks command, specify the driver, classpath, and URL in the properties file. You can also specify these properties in your command line.

Then run the release-locks command:

liquibase release-locks --changelog-file=example-changelog.xml

Command arguments

Attributes Definition Requirement
--url Required
--changelog-file Optional
--default-catalog-name Optional
--default-schema-name Optional
--driver Optional
--driver-properties-file Optional
--password Optional
--username Optional
Output

When successful, the release-locks command produces the following output:

Liquibase Community 4.9.1 by Liquibase
Successfully released all database change log locks for 'DBUSER@jdbc:h2:tcp://localhost:9090/mem:dev'
Liquibase command 'release-locks' was executed successfully.
