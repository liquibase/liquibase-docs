init hub

The init hub subcommand connects your local activity to with an API key and sets up the environment to use .

Uses

If you are new to , the init hub subcommand will create and add project files for you by calling the init project subcommand.

If you already use and want to connect to the existing files with , the init hub subcommand detects the and properties files with the help of the --project-dir parameter, and registers the to a Project.

Syntax

From the command prompt or terminal, run the following command and follow the CLI instructions:

liquibase init hub --token=<the token you see in the Liquibase Hub Getting Started dialog>

To connect your existing project files, add the --project-dir parameter:

liquibase init hub --token=<the token you see in the Liquibase Hub Getting Started dialog> 
--project-dir=<the/path/to/a/directory>
