init

The init command sets up configuration files and parameters required to run and access a database. The command works with subcommands for specific functions – copying project files to a new directory, connecting to , setting up a project, and starting the H2 in-memory database. ( 4.7.0 and later versions).

Subcommands

Command Description
init copy
init hub
init project
init start-h2
