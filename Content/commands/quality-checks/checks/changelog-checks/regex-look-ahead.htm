RegexLookAhead

The RegexLookAhead check is a pre-written Regex quality check that allows to look for a positive pattern based on two user-provided inputs. This check is pre-configured so users do not have to hand write regex's to look forward in their .

This check triggers when the user-supplied regular expression for a primary match is followed by a user-supplied regex 'lookahead' match.

Uses

The RegexLookAhead check can be used to confirm conditions within the . This check allows you to check for specific patterns in your and then produce a logged record of changeset structure when certain conditions are met. When it is executed successfully the checks message is logged to the console and logs as proof that changesets conform to specific standards.

Run the RegexLookAhead check

To run the RegexLookAhead check, ensure you have a Liquibase Pro license and the checks-scope property includes changelog. The checks-scope parameter is set in the default properties file, environment variable, or any standard method.
