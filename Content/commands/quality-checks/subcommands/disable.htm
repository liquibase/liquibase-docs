disable

The disable subcommand disables a specific check. You can specify which check to disable using the --check-name parameter.

Uses

You can typically use the disable command to stop executing a check you no longer require. The liquibase checks run command does not evaluate disabled checks when you execute it.

Syntax

Run the command specifying your values:

liquibase checks disable --check-name=ChangesetCommentCheck

The command disables the check in the checks settings file.

Parameters

  • CLI
  • Flow
  • liquibase.properties
  • JAVA_OPTS
  • Environment variable
