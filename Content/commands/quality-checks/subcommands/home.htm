Quality Checks Sub Commands

The liquibase checks command is a command group with sub commands that let you customize and execute quality checks. It allows you to improve the quality of your database changes.

  • Sub commands require specific parameters noted in the table.
  • Customizations to checks configuration are stored in the checks settings file.

checks subcommands list

If you use Maven, see Maven checks.show and Maven checks.run.

To validate that multiple types of objects in your database adhere to your naming conventions, use the Multiple Objects Quality Check. You can also format the output of your quality checks as JSON: see Quality Checks JSON Object. For more information about using quality checks, see Quality Checks.
