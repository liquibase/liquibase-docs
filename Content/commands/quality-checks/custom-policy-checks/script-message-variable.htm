Script Message Variables for Custom Policy Checks

When you set the script message in a custom policy check you have the option to create variables within your Python script. This guide shows where those are set up within a script and how can call on those variables.

We recommend you review this set of Python script examples to set up full messages or messages with replaceable strings.

Prerequisites

This guide picks up the Option for advanced users step on the Custom Policy Checks page. Refer to that page for setting up custom policy checks.

Step-by-step guide

  1. In the CLI set the script message. This message will display when the check is triggered. In this example we will set the message to:
    Identifier __ID_NAME__ should not include quotes.

  2. Your Python script will already contain the status message. This status message can contain a replaceable string that is identified by . In this example, the replacable string is __ID_NAME__ :
    status_message = "Identifier __ID_NAME__ should not include quotes."

    sees this replaceable string and refers to the Python script to find out what to do with it.

    status_message = str(script_helper.git_gitscript_message()).replace(__ID_NAME__", identifier)
    will pull the __ID_NAME__ and replace it.
