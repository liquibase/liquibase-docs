rollbackOneChangeSetSQL command


The rollbackOneChangeSetSQL command is a helper command that allows you to inspect the SQL will run to revert the specified in the rollbackOneChangeSet command. It is only available for users.

Uses

The rollbackOneChangeSetSQL command is typically used when you want inspect the raw SQL that uses to revert your when you run the rollbackOneChangeSet command, so that you don’t unintentionally make a mistake.

Running the rollbackOneChangeSetSQL command

Before running the rollbackOneChangeSetSQL command, gather the following information from your table or by running the history commands:

  • The Author of the you want to revert
  • The ID of the you want to revert
  • The file name ( path) of the you want to revert

Then run the rollbackOneChangeSetSQL command, with your information:

For more command specific help, type liquibase rollbackonechangesetSQL --help into the command prompt.
