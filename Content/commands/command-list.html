flow commands

Command Description
flow

Creates portable, platform-independent workflows that can run anywhere without modification. Flow allows you to put multiple commands in a single file, called a Flow File, so they can all run sequentially by typing a single command into the CLI. The flow command provides the convenience of performing all of the steps in one place.
flow validate

Validate a series of commands contained in one or more stages, as configured in a flow file.
