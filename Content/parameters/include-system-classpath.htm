include-system-classpath

The include-system-classpath parameter is a Boolean that defines whether includes the system classpath when resolving classes at runtime. The default value is true.

Uses

The classpath refers to locations searched for drivers. always includes:

  • Locations in the classpath setting
  • The current directory
  • Files in the liquibase/lib and liquibase/internal/lib directories

When you use the include-system-classpath parameter set to true, also includes locations configured within the underlying Java system.
