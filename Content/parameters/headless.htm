headless

The headless parameter is a Boolean value that allows you to force to run without access to the console for input and output. By default, the value is false.

Uses

You can use the headless parameter when you run in automation or with CI/CD tools. For example, you may need to skip the prompt for running the init project command in interactive mode. In this case, you can set the headless parameter to true to run commands without interaction.

If you run in containers, the headless parameter will prevent from trying to access the display, keyboard, or mouse.
