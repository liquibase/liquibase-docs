ui-service

--ui-service is a global parameter that specifies the default UI service logger used by . Valid values are CONSOLE and LOGGER. The default value is CONSOLE. This parameter is available in 4.25.0 and later.

Uses

If you set --ui-service to CONSOLE, sends output messages to the console (STDOUT) by default.

However, you may want to send output messages to the log by default, such as a third-party logging tool like Log4j or Spring Boot's service logger. To allow this, set --ui-service to LOGGER.
