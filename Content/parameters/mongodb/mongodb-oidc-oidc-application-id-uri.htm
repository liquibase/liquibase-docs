mongodb-oidc-oidc-application-id-uri

--mongodb-oidc-oidc-application-id-uri is a global parameter for the Liquibase MongoDB Pro extension. It specifies the application ID URI used for OIDC authentication in cloud providers. For example, with Azure: api://8d719796-c744-4352-aab9-47a52c363907.
