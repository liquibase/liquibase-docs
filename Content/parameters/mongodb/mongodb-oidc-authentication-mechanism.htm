mongodb-oidc-authentication-mechanism

--mongodb-oidc-authentication-mechanism is a global parameter for the Liquibase MongoDB Pro extension. It specifies the authentication mechanism for Liquibase to use with MongoDB. Valid values are: MONGODB-OIDC. If no value is specified, Liquibase tries to connect using a username and password.

Specifying this parameter is required for OIDC authentication. It is available in the MongoDB extension 1.5.0 and later. For more information, see MongoDB: Enterprise Authentication Mechanisms.
