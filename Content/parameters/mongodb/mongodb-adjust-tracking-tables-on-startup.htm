mongodb-adjust-tracking-tables-on-startup

--mongodb-adjust-tracking-tables-on-startup is a global parameter for the MongoDB Open Source extension. It specifies whether to validate the  DATABASECHANGELOG and DATABASECHANGELOGLOCK collections on startup. If they are not up to date with the current release, adjusts them. The default value is true.

Uses

It is a best practice to keep this parameter enabled whenever you upgrade to a new version.
