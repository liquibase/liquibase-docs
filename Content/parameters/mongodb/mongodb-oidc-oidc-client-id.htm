mongodb-oidc-oidc-client-id

--mongodb-oidc-oidc-client-id is a global parameter for the Liquibase MongoDB Pro extension. It specifies the Client ID of the User-Assigned Identity for OIDC authentication.

Specifying this parameter is optional for OIDC authentication. It is available in the MongoDB extension 1.5.0 and later.
