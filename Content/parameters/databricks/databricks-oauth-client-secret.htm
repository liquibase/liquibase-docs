databricks-oauth-client-secret

--databricks-oauth-client-secret is a global parameter for the Databricks extension. It specifies the Client Secret for OAuth authentication on Databricks. The Client Secret is the service principal's OAuth Secret value.

Specifying this parameter is required for OAuth authentication. It is available in the Databricks extension 1.0.0 and later.
