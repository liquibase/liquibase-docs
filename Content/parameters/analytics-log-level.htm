analytics-log-level

--analytics-log-level is a global parameter. You can use it to specify the amount of detail provides in user-facing logs for Anonymous Analytics. It is available in 4.30.0 and later. The default value is INFO.

--analytics-log-level does not control what analytic data collects. To enable or disable analytics, use --analytics-enabled.

Uses

--analytics-log-level controls the detail level of analytics logs displayed to the user.
