dynamodb-waiters-fail-on-timeout

--dynamodb-waiters-fail-on-timeout is a global Boolean parameter for the DynamoDB Pro extension. It specifies whether fails if the DynamoDbWaiter times out while attempting to verify whether a resource like a table in your DynamoDB database is in a valid state. The default value is true.

Uses

The --dynamodb-waiters-fail-on-timeout parameter controls which exit code  returns if the waiter times out:

  • If you want to treat a timeout like a significant error and stop running, leave the parameter at its default value of true. This way, returns an exit code of 1.
  • If you want to be informed of a timeout but want to continue running, set the parameter to false. This way, returns an exit code of 0.
