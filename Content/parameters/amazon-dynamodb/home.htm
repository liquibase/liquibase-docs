Liquibase Parameters for Amazon DynamoDB Pro

You can control the way Liquibase behaves when running commands on your Amazon DynamoDB database by setting Liquibase parameters. For example, you can use them to configure timeout values and data billing settings.

For a list of general Liquibase parameters, see [%=General.Liquibase%] Parameters.

For more information about Amazon DynamoDB, see [%=General.Liquibase%] Amazon DynamoDB Pro Extension Overview. For a getting started tutorial, see Using [%=General.Liquibase%] with Amazon DynamoDB Pro.
