custom-log-data-frequency

The --custom-log-data-frequency global argument lets you specify how frequently to include custom data in your s. Valid values are ONCE and REPEATED. The default value is REPEATED. It is available in 4.24.0+. Learn more about custom data: Custom and Private Data in [%=General.StructuredLog%]s.

Uses

By default, 4.24.0+ includes all custom data in your s, so the same piece of data may appear multiple times in logs. This enables you to use dashboard queries and reports that depend on these custom key:values pairs. However, if you only want to include your custom data once per log, you can set --custom-log-data-frequency to ONCE.
