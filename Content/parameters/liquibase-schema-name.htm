liquibase-schema-name

The liquibase-schema-name global parameter is a string that specifies the schema to use for the creation of Liquibase objects, like the and tracking tables.

Uses

If you are working in a multi-schema project, you can use liquibase-schema-name to separate the and tables from other Liquibase objects. This way, you will have a tracking schema and one or more managed schemas.

If you need to work in a test schema for your project, you can set liquibase-schema-name to control where Liquibase looks for the tracking tables during test runs.
