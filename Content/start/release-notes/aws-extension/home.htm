AWS Extension 1.0.0 - February 17, 2025

4.31.0+: The AWS extension consolidates the three AWS S3, AWS Secrets Manager, and Amazon DynamoDB Pro extensions into one. This single, convenient source provides a stable, simplified user experience and prevents conflicts between the three tools.

From now on this extension replaces all individual AWS extensions.

For more information, see Integrate AWS with [%=General.LBPro%] and Install the [%=General.LBPro%] AWS Extension

Amazon Dynamo DB extension

Prior to the release of the AWS extension 1.0.0, the Amazon Dynamo DB extension was an individual extension. As of February 2025 Amazon Dynamo DB extension is included in the Install the [%=General.LBPro%] AWS Extension.
