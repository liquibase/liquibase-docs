[PRO] Observability Initiative - Structured Logging

Structured Logging makes operation data easily available for automated monitoring and analysis tools to read, query, and act upon in automated workflows. This feature significantly improves the way logs its actions and events to make records machine-readable, easily-ingested, and queryable by industry-standard observability and analysis tools.
