Notable Changes

New liquibase flow command

  • liquibase flow and new flow-file (DAT-10419) by in #2946
  • The liqubase flow command and the Flow File on which it operates allow the user to place multiple commands within a single file so they can all run sequentially with a single command into the CLI. This is currently a restricted capability, so please contact mario@liquibase.com to request a license key. Learn more at flow
