Notable Changes

Liquibase 4.29.1 .zip files: Restored to standard directory structure

  • This release correctly places Liquibase .zip files inside liquibase-4.29.1 directory to match the pre-4.29.0 directory structure.

BigQuery Extension: using with Google Big Query

  • With Liquibase 4.29.1, Google BigQuery is now supported as an "out-of-the-box" database enabling users to more quickly achieve value. Also included is support for features such as Flows, Policy Checks, and Operation Reports with Google BigQuery so that users will have the efficiency and productivity benefits of when working with the platform.
