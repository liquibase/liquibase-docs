4.8.0 Release Notes

Full Changelog: v4.7.1...v4.8.0 (February 22, 2022)

This release includes the init hub subcommand, support for MSSQL Server sqlcmd utility, community-contributed pull requests, and and bug fixes.

Notable Changes

4.8.0 introduces the following functionality:

  • The init hub subcommand that connects your local activity to and sets up the environment to use . [DAT-8769]
  • [PRO] The sqlcmd utility support to process complex SQL for MSSQL Server. provides the liquibase.sqlcmd.conf file to pass arguments to your executor when running . [DAT-7447]

    • For more information, see Use Native Executors with Microsoft SQL Server.

  • Changes to the behavior of the XML parser, which no longer allows referencing external DTD files for security reasons. If you use externally defined entities or any other potentially insecure XML feature in your s, set liquibase.secureParsing=false. [PR#2384] [LB-2218]

    • For more information about the ways to set the parameter, see [%=General.Liquibase%] Parameters.

  • The upgrade of the postgresql (from 42.2.12 to 42.3.2) and h2 (from 2.0.206 to 2.1.210) drivers that includes in the installation package. If you use those drivers and upgrade an existing installation, remove the earlier versions of drivers from the LIQUIBASE_HOME/lib directory.

Enhancements

  • Implemented the SimpleObjectConstructor interface for DB2 on z/OS [DAT-8580]
  • Included the CLI instructions on how to use the properties file with a nonstandard name when running the init project subcommand [DAT-9041]
  • Improved the output message for init start-h2 when the H2 database driver is specified, but there is no connection detected [DAT-8992]
  • Added validation errors for the enableCheckConstraint, disableCheckConstraint, dropPackage, dropPackageBodys [DAT-9017]
  • [PR#2367] [Mike Olivas] Added example rollback scripts to the example-changelog.sql file [LB-2220]
  • [PR#1648] [Daniel Gray] Improved the exception error message for the customChange node with no class [LB-1144]
  • [PR#2222] [msimko81] Added the offline mode support for the rollback-sql <tag> operation [LB-2198]
  • [PR#2273] [Tsvi Zandany] Added the autocomplete quality checks commands for macOS
  • [PR#2308] [Valentin Blistin] Added the close method for the ClassLoaderResourceAccessor class [LB-2205]
