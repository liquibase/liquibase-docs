MongoDB Extension Release Notes

Release notes for the MongoDB open-source software (OSS) and MongoDB Pro extensions.

The release of the liquibase-mongo extension is independent of liquibase-core releases. Versions numbers of this extension and liquibase-core are not necessarily synchronized. See the compatibility chart for more information.

MongoDB OSS 4.23.1 and MongoDB Pro 1.2.0 – September 6–12, 2023

The MongoDB OSS and Pro extension are compatible starting with 4.23.1, and are not backwards compatible with previous versions. For users of core versions between 4.19.0 and 4.21.1, please use the previous 4.x Mongo extension release as shown in the compatibility matrix.

MongoDB OSS 4.23.0 and MongoDB Pro 1.1.1 – July 11–13, 2023

The MongoDB OSS and PRO extension are major version releases compatible starting with 4.23.0, and are not backwards compatible with previous versions.

For users of core versions between 4.19.0 and 4.21.1, please use the previous 4.x Mongo extension release.
