Databricks 1.3.0 Release Notes

Databricks 1.3.0 is a major release. It was released on August 20, 2024. See GitHub: Databricks v1.3.0.

For documentation and guidance on using with Databricks, see: Using [%=General.Liquibase%] with Databricks Platforms.

Notable Changes

  • Ability to set/unset/alter TBLPROPERTIES key values
  • Now compatible with  connect command
  • Now compatible with  [%=General.databasechangeloghistory%] table feature
