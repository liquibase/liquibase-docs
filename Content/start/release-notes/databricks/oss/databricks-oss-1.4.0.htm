Databricks 1.4.0 Release Notes

Databricks 1.4.0 is a major release. It was released on November 27, 2024. See GitHub: Databricks v1.4.0. Full Changelog: liquibase-databricks-v1.3.0...v1.4.0.

For documentation and guidance on using with Databricks, see: Using [%=General.Liquibase%] with Databricks Platforms.

Notable Changes

  • Fix bugs and fill gaps associated with basic support of TABLE and VIEW object properties
  • Remove inconsistencies and gaps from Database Inspection commands on TABLE and VIEW objects
