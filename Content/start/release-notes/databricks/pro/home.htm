Pro Databricks 1.0.0: February 6, 2025

Databricks 1.0.0 is a major release. It is the initial release of the extension for Databricks, which expands the functionality of the extension. It contains new features like TABLE object versioning, FUNCTION stored logic support, VOLUME object support, OAuth authentication, and more.

For more information, see [%=General.LBPro%] Databricks 1.0.0 Release Notes.
