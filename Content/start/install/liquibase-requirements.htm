Requirements

is designed to run with minimal hardware and software.

Hardware

  • Disk space: 300 MB
  • Processor:
    • Intel Core i3
    • ARM processor
    • 1 CPU
  • Memory: 1 GB RAM

Software

  • Any version of Windows, macOS, or Linux/Unix with a supported Java installation.
  • Java 8 is the minimum required version, although we recommend using LTS versions 11 and 17. Our team regularly tests on these versions and Liquibase ships with 17.
    • If you use the Installer, Java is included automatically.
    • If you install manually, you must also install Java, which is available from Adoptium. If you use the Installer, this version of Java is used during installation.
