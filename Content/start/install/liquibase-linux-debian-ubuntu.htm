Install on Linux with Debian/Ubuntu

You can install, upgrade, and uninstall on your Linux machine with the Debian/Ubuntu package installer by following the steps on this page.

Install

To install using the Debian/Ubuntu installer package, follow the steps below:

  1. Open a terminal.
  2. Run the following command to import the GPG key and add the repository to the apt sources list:
    3. wget -O- https://repo.liquibase.com/liquibase.asc | gpg --dearmor > liquibase-keyring.gpg && \
cat liquibase-keyring.gpg | sudo tee /usr/share/keyrings/liquibase-keyring.gpg > /dev/null && \
echo 'deb [arch=amd64 signed-by=/usr/share/keyrings/liquibase-keyring.gpg] https://repo.liquibase.com stable main' | sudo tee /etc/apt/sources.list.d/liquibase.list && \
sudo apt update && sudo apt install liquibase
  3. Update the package lists:
    4. sudo apt-get update
  4. Install :
    5. sudo apt-get install liquibase

is now installed on your system.

Optional: install a specific version

To install a specific version of using the Debian/Ubuntu installer package, you can use the package manager's version pinning mechanism. Run the following command and replace x.y.z with the desired version number:

sudo apt-get install liquibase= x.y.z

The specified version of is now installed on your system.

Upgrade

To upgrade to the latest version, follow these steps:

  1. Open a terminal.
  2. Update the package lists:
    3. sudo apt-get update
  3. Upgrade :
    4. sudo apt-get upgrade liquibase

has now been upgraded to the latest version available in the repository.

Uninstall

To uninstall from your system, follow these steps:

  1. Open a terminal.
  2. Remove the package:
    3. sudo apt-get remove liquibase

is now uninstalled from your system.
