Install on Linux with Red Hat/CentOS

You can install, upgrade, downgrade, and uninstall on your Linux machine with the Red Hat/CentOS package manager by following the steps on this page.

Liquibase supports all versions of Linux and Unix that use a supported Java installation. Visit the Requirements page for more information.

Install

To install using the Red Hat/CentOS package manager, follow the steps below:

  1. Open a terminal.
  2. Import the public key (as root):
    3. rpm --import https://repo.liquibase.com/liquibase.asc
  3. Add the remote repo to yum (as root):
    4. yum-config-manager --add-repo https://repo.liquibase.com/repo-liquibase-com.repo
  4. Install :
    5. yum install liquibase

is now installed on your system.

Optional: installing a specific version

To install a specific version of using the Red Hat/CentOS package manager, you can use the package manager's version specification mechanism. Run the following command and replace x.y.z with the desired version number:

yum install liquibase-x.y.z

The specified version of will be installed on your system.

Upgrade

To upgrade to the latest version, follow these steps:

  1. Open a terminal.
  2. Update to the latest version:
    3. yum update liquibase

will be upgraded to the latest version available in the repository.

Downgrade

To downgrade to a previous version, follow these steps:

  1. Open a terminal.
  2. Downgrade to the previous version:
    3. yum downgrade liquibase

will be downgraded to the previous version available in the repository.

Uninstall

To uninstall from your system, follow these steps:

  1. Open a terminal.
  2. Remove the package:
    3. yum remove liquibase

will be uninstalled from your system.

If you encounter any issues or have further questions, please refer to the documentation or seek support from the community.
