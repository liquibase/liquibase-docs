Install with Chocolatey

You can install using the Chocolatey package manager by following the steps on this page.

This package manager makes the release process of packages as easy and fast as possible so it will stay close to the release pipeline.

Prerequisites

  1. Ensure you have the necessary permissions (such as administrator rights) to install, upgrade, or remove software on your system.
  2. Install Chocolatey on your machine.

Installing with Chocolatey

  1. Open Command Prompt or PowerShell: Open Command Prompt or PowerShell with administrator privileges. You can do this by right-clicking on the Command Prompt/PowerShell icon and selecting Run as administrator.
  2. Install : Type the following command and press Enter:
    3. choco install liquibase
  3. Verify Installation: Once the installation is complete, you can verify your installation by running:
    4. liquibase --version

    This command displays the version installed on your system.

Upgrading with Chocolatey

  1. Check for Updates: Before upgrading, it's a good practice to check if there are updates available for . Run the following command:
    2. choco upgrade liquibase --noop

    This command simulates the upgrade process without actually performing it.

  2. Upgrade : If there are updates available and you're ready to upgrade, run:
    3. choco upgrade liquibase
  3. Verify Upgrade: After the upgrade process is complete, verify that has been upgraded successfully by running:
    4. liquibase --version

    This command displays the version installed on your system.

Removing with Chocolatey

  1. Uninstall : To remove from your system, run the following command:
    2. choco uninstall liquibase

    Confirm the uninstallation when prompted.

  2. Verify Removal: After the uninstallation process is complete, verify that has been removed by running:
    3. liquibase --version

Continuous build

The build and deploy process is triggered from liquibase/liquibase every time a new version is released. Check deploy-package.yml for more details.
