Using with PostgreSQL Platforms

PostgreSQL is an open-source, object-relational database system that supports a large part of the SQL standard and offers many modern features. For more information, see the PostgreSQL documentation page.

You can use with several PostgreSQL database management systems, including PostgreSQL, PostgreSQL AWS RDS, Postgres AWS Aurora, Google Cloud SQL, and Azure Database on Postgres.
