Use with PostgreSQL

PostgreSQL is an open-source, object-relational database system that supports a large part of the SQL standard and offers many modern features listed below. This guide covers how to setup with PostgreSQL. For more information, see the PostgreSQL documentation page.

Supported versions

Deprecated Versions

  • PostgreSQL Server 11 is deprecated as of November 9, 2023.

    • support for this version ends on February 7, 2024.
    • Other versions remain supported.

Prerequisites

Install drivers

CLI users

To use and PostgreSQL, you need the JDBC driver JAR file (Maven download).

<dependency>
    <groupId>org.postgresql</groupId>
    <artifactId>postgresql</artifactId>
    <version>42.6.0</version>
</dependency>

Also, you can use the PostgreSQL Extension JAR file, which is the vacuum extension that adds an additional tag or command to support vacuuming.

Establish a connection

  1. Ensure your PostgreSQL database is configured. Check the status by running the pg_isready command. For more information about the options you can specify when running the command, see the pg_isready webpage.
