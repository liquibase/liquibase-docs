Use with Amazon Aurora PostgreSQL

PostgreSQL is a cloud based, object-relational database system that is fully compatible with MySQL and PostgreSQL.

This guide covers how to setup with Amazon Aurora.

For more information, see Working with Amazon Aurora PostgreSQL

Install drivers

CLI users

To use and PostgreSQL, you need the JDBC driver JAR file (Maven download).

<dependency>
    <groupId>org.postgresql</groupId>
    <artifactId>postgresql</artifactId>
    <version>42.6.0</version>
</dependency>

Also, you can use the PostgreSQL Extension JAR file, which is the vacuum extension that adds an additional tag or command to support vacuuming.

Configure connection

  1. Ensure your PostgreSQL database is configured. Check the status by running the pg_isready command. For more information about the options you can specify when running the command, see the pg_isready webpage.
