How To: Use Oracle's Rollback DML (Using Change Set Wizard)

extends Oracle's Rollback DML capability. This document shows you steps on how to configure your project with this capability.

  1. has a new project setting called Forecast Data Modification Changes. This is available in Settings tab:

  2. Launch the Change Set Wizard

  3. Select Oracle Utilities > Execute with SQLPlus and click Next

  4. Browse and select a DML script, then click Next

  5. Review the summary script. If this is the only script you are adding to the change set then click Finalize. (If adding additional script, click Next and repeat steps 3 thru 5)

  6. Provide ID and Author information as well as any additional information, such as Label information. Then click Finish. Your change set is ready for forecast and deploy operations.

  7. The Forecast DML feature is used during the Forecast operation. Click on the "Forecast" button for your target database

  8. Provide the correct labels (you specified in Step 6) and click Finish
