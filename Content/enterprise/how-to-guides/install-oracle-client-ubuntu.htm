How To: Install Oracle client on Ubuntu 16.04

This guide is intended to install the Oracle full client on Ubuntu without an X session.

Caveat: this is not really a full supported configuration by Oracle

install the full client (not the instant client) from http://download.oracle.com/otn/linux/oracle12c/121020/linuxamd64_12102_client.zip for version 12.1

According to https://ubuntuforums.org/showthread.php?t=1961644, you need to add some packages

sudo apt-get install -y libaio1
sudo apt-get install -y libaio-dev
sudo apt-get install -y unixODBC
sudo apt-get install -y unixODBC-dev
sudo apt-get install -y expat
sudo apt-get install -y sysstat
sudo apt-get install -y libelf-dev
sudo apt-get install -y elfutils
sudo apt-get install -y lsb-cxx
sudo apt-get install -y pdksh
sudo apt-get install -y libstdc++5
sudo apt-get install -y ia32-libs
sudo apt-get install -y ksh
sudo apt-get install -y lesstif2
sudo apt-get install -y alien
sudo apt-get install -y gcc
sudo apt-get install -y gawk
sudo apt-get install -y binutils
sudo apt-get install -y gawk
sudo apt-get install -y x11-utils
sudo apt-get install -y rpm
sudo apt-get install -y alien
sudo apt-get install -y lsb-rpm
sudo apt-get install -y libmotif3
sudo apt-get install -y lesstif2
sudo apt-get install -y openssh-server

and some links:

sudo ln -s /usr/bin/basename /bin/basename
sudo ln -sf /bin/bash /bin/sh
sudo ln -s /usr/bin/rpm /bin/rpm
sudo ln -s /usr/bin/awk /bin/awk
sudo mkdir -p /usr/lib64
sudo ln -s /usr/lib/x86_64-linux-gnu/libc_nonshared.a /usr/lib64/ 
sudo ln -s /usr/lib/x86_64-linux-gnu/libpthread_nonshared.a /usr/lib64/ 
sudo ln -s /usr/lib/x86_64-linux-gnu/libstdc++.so.6 /usr/lib64/ 
sudo ln -s /lib/x86_64-linux-gnu/libgcc_s.so.1 /lib64
sudo ln -s /usr/lib/i386-linux-gnu/libpthread_nonshared.a /usr/lib/libpthread_nonshared.a

Now run the installer in silent mode (see an example response file client_install.rsp)

unzip linuxamd64_12102_client.zip
cd client
./runInstaller -silent -responseFile /path/to/client_install.rsp
