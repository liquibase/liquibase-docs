Use a Proxy Server with

requires functionality from plugins. If a firewall blocks you from accessing sites for the plugins directly, you may be able to use a proxy server.

Information Needed

Contact your IT organization and ask them for the following information for the corporate proxy server. Host and Port are always required. If authentication is required, you also provide a User and Password.

HTTP Proxy

  • Host
  • Port
  • (optional) User
  • (optional) Password

HTTPS Proxy

  • Host
  • Port
  • (optional) User
  • (optional) Password

Once you have this information, follow the instructions below to setup the proxy.

  1. In the menu, select the File > Preferences menu item



  2. Click the triangle next to General, then click Network Connections.



  3. For Active Provider, select Manual.
  4. Select HTTP (in Proxy entries list)
    1. Click Edit.
    2. Enter values for Host and Port.
    3. Optional: Select Requires Authentication and enter values for User and Password.



  5. Select HTTPS (in Proxy entries list).
    1. Click Edit.
    2. Enter values for Host and Port.
    3. Optional: Select Requires Authentication enter values for User and Password.



  6. Click OK.



  7. Your proxy is now set up.
