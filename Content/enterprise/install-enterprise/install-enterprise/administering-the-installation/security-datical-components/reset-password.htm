Resetting the Password for the Update Site

You can reset your credentials in the GUI using the following steps.

Remove the Existing Password

  1. In the GUI, select File > Preferences.



  2. In the left panel of the Preferences window, open General.



  3. Open Security.



  4. Click Secure Storage.



  5. Click the Contents tab.



  6. Click Default Secure Storage, then click a default storage entry, and click Delete.



  7. When you are prompted to confirm the deletion, click Yes.



  8. When you are prompted to restart, click click Yes.



Set the New Password

  1. After Datical restarts, select Help > Install New Software.



  2. When you see Authentication failed dialog, click OK to continue.



  3. In Work with, select http://update.datical.com.



  4. Provide your username and password, select Save password, then click OK.



  5. Click No to skip password recovery.



  6. The update.datical.com repository is now available.

Note: Starting with release 3.72, the new update site located at http://update.datical.com contains everything in one place. You can simplify your install/update by removing the old sites and only connecting to the new one.
