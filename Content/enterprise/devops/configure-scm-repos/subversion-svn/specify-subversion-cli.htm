Specify a User Name when Running Subversion on the Command Line

Shared environments are often set up on Linux.

If you are working in a shared environment, you may need to specify your user name when running Subversion from the command line. To do this, use the --username option in the commands you run.

Example
svn commit --username nbk5k9e -m "adding additional options to datical_deploy script, also further enhanced ecom library scripts"
