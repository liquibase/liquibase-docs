Add a Project to Subversion

Share the Project with the SVN Repository

  1. Right-click on a project, then select Team > Share Project.



  2. Select the Create new repository location radio button and click Next
  3. Enter the URL to your SVN repository and click Next

  4. Share the project in the SVN repo
    1. Select Use specified folder name.
    2. Click Select to browse for repo.
    3. Click Finish



  5. Enter the initial import comment and click Finish

  6. The project is now associated with the SVN Repository

Configure the Subversion Ignore File

  1. Right-click a project, then select Team > Set Property.

  2. Add the following values, then click OK.

    Property name:
     svn:ignore
    Property content:
    Logs
    Reports
    Snapshots
    Profiles
    .project
    .classpath
    .metadata
    .reporttemplates
    daticaldb.properties
    daticaldb*.log
    *.zip

Commit the Project to Subversion

Commit the project files to the trunk.

  1. Right-click the project, then select Team > Commit.



  2. Enter a comment, thenclick OK.
