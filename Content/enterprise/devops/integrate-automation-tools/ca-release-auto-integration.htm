CA Release Automation Integration

Download the DaticalDB4CARA Plug-in

  1. Go to URL: https://github.com/Datical/DaticalDB4CARA/tree/master/dist.
  2. Select the latest .jar file
  3. Select the View Raw button to download the file.

Install the Plug-in

  1. In CA Automation Studio, go to Administration > Action Management.
  2. Select the Green Plus image on the right.
  3. Navigate to the location of DaticalDB4CARA Zip file and select the file.
  4. Select Open. (Default username and password is superuser/suser.)

Create Components to Set Properties for

Custom Actions all use the DaticalDBLocation and DaticalDBProjectDirectory parameters.

  1. View the properties in the Parameters list (tab located on the right side of the CA RA UI).

  2. Create components to set each of the properties.




Use Properties in Flow Steps

In subsequent steps, you can then use the properties.



Add Components for Automation Flow

  1. Complete integration by adding more components to run operations within CA Release Automation.
    • The daticalDBCmd and daticalDBDriversDir are already populated.
    • Enter the path of your Project Directory relative to your project top level folder. If you checked your Project Directory into the root, use "." (period).
  2. Configure further steps as needed.
