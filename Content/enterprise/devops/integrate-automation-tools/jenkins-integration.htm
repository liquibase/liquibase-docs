Jenkins Integration

The following sections detail how to operate the CLI to retrieve status about the servers and ChangeLog, Forecast, Deploy, and Rollback

View DaticalDB4Jenkins Plugin Information

The plug-in is maintained through Jenkins.

See the Jenkins DaticalDB4Jenkins page.

Install the DaticalDB4Jenkins Plug-in

  1. Open the Jenkins web interface.
  2. Select Manage Jenkins > Plugin Manager > Available Plugins.
  3. Enter "datical" into the filter.
  4. Check the DaticalDB4Jankins plugin, then clickInstall without restart.



Set Paths for the Plug-in

  1. Select Manage Jenkins > Configure System.
  2. Scroll down until you see the settings.

  3. Enter paths for .
    • Command Directory - Fully qualified path to the hammer script. Important: include the script name (hammer.bat for Windows systems or hammer.sh for Linux systems).
    • Database Drivers Directory - Fully qualified path for the database drivers used by .

  4. Click Save.

Add Steps to Jenkins Projects

  1. Select the Project where you want to add to add a Datical step
  2. Click Configure.
  3. Add a step. Configure with the information for your environment.
  4. Click Save when done.
