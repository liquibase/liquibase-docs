Installing Postgres EDB Clients

  • uses the following client and Postgres Command Line Utilities:

    • edbplus

    • pg_dump

    • pg_restore

  • Ensure that the desired version of the above are installed.

  • Ensure that the installation directories are in the PATH environment variable

  • The edbplus client is included with installation in <Datical_InstallDir>/edbplus folder. This folder needs to be added to PATH environment variable.

  • The latest version of the Postgres Command Line Utilities is backwards compatible to EDB Postgres 9.6

  • If using EDB Postgres 9.5 or earlier, ensure your version of Postgres Command Line Utilities matches that of the database

  • If packaging runs and fails, a .DMP file is left on the client host.
