Install SQL Server Clients

Make sure the clients are installed and the installation directories are in the PATH.

  • Download and install the ODBC driver
    https://www.microsoft.com/en-us/download/details.aspx?id=36434

  • Download and install SQLCMD (sqlcmd.exe)

    • Windows: https://www.microsoft.com/en-us/download/details.aspx?id=36433

    • Linux: https://docs.microsoft.com/en-us/sql/linux/sql-server-linux-setup-tools?view=sql-server-ver15
