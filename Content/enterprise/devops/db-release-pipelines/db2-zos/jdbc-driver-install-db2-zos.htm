JDBC Driver Installation for DB2 on zOS

Overview

DB2 on z/OS requires that a specific JDBC driver and corresponding license file is setup in your environment to enable connectivity to the mainframe.

Installing the License File and Driver

To use the DB2 on zOS license file, you choose between the following two options:

  1. Set up an environment variable as: DATICAL_DB2Z_LICENSE_JAR
  2. Put the license file into the installation directory as: <installation-dir>/repl/lib

Follow these

Copy the following two jar files into the appropriate folders within your install directory:

  • Copy the DB2 on z/OS JDBC Driver (db2jcc4.jar) to the folder: <install_dir>/plugins
  • Copy the DB2 on z/OS License File (db2jcc_license_cisuz.jar) to the folder: <install_dir>/repl/lib or Set up a license file environment variable as: DATICAL_DB2Z_LICENSE_JAR

Desktop GUI - Driver Configuration

  1. Remove the Old DB2 Driver
    • If the original DB2 JDBC Driver has been installed in , remove it from the plugins folder.
    • remove file: <install_dir>/plugins/com.datical.db.drivers.db2_1.0.11.20181126111708.jar
  2. Reference the DB2 Driver in the Preferences.
    • From the menu, select File → Preferences.

    • Navigate to the JDBC Drivers → IBM DB2 Drivers preferences page.

    • Click the Browse button and select the JDBC jar file (db2jcc.jar) in the <install_dir>/plugins directory.

    • Click the Validate button and make sure you see the "Successfully instantiated driver!" response.
      • Click Apply.
      • Click OK to close the dialog.

  3. Test the Connection
    • Assuming that you already have a database connection setup, click the Status button and verify that the Database Drum is Green and no errors appear.

CLI - Driver Configuration

  1. Create a liquibase.properties file (in a project directory) and add the following line pointing to the directory where you added the driver file (in step #1 above).
    • drivers=<install_dir>/plugins
  2. Test the database connection via the CLI
    • hammer testconnect DEV
