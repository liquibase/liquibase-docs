CLI Commands Overview

hammer provides a command-line interface. This document describes the commands and options available.

Running Commands with Hammer

To run a command directly from a system prompt, use the hammer command followed by the command.

hammer [command] [options]

- Required arguments are shown in angle brackets: <required argument>

- Optional arguments are shown in square brackets: [optional argument].

- Case-insensitive - Command names and options can be in any case or combination. (Version 5.9 and later)

See CLI Commands and Command Options for more information.

Commands for Showing and Setting Project Settings

The set and show commands work on project settings.

For more information:

See Configuring Project Settings. It contains a list of settings and where you can view and set them (GUI, CLI).

See entries for set and show in CLI Commands.

For usage information run:

hammer set help

hammer show help
