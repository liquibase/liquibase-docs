Upgrade Liquibase Enterprise

If you use in conjunction with , it is essential that you first upgrade first and then upgrade for the improvement to be successful.

Prerequisites

  • You have downloaded the installer for Windows or Linux systems. See Downloading the Installer

  • Prior to upgrading, you can choose to keep your older version installed or uninstall it.

  • If you choose to uninstall, follow these instructions: Uninstalling Components.

  • If you choose to keep your older version installed, follow this best practice:

BEST PRACTICE: It is always best to maintain a consistent naming convention for directory where you install Liquibase Enterprise.

Your latest version could be installed in C:\Apps\DaticalDB directory on Windows or /opt/DaticalDB directory on Linux and rename directory for older versions as follows:

  • C:\Apps\DaticalDB_7.0 on Windows or /opt/DaticalDB_7.0 on Linux

  • C:\Apps\DaticalDB_7.2 on Windows or /opt/DaticalDB_7.2 on Linux

  • C:\Apps\DaticalDB_7.5 on Windows or /opt/DaticalDB_7.5 on Linux
