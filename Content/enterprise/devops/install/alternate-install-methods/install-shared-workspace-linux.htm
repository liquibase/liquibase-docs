Installation on a Shared Workspace on Linux

Review the instructions in Accessing Remote Linux Systems as needed.

As root, install into a shared install directory on linux to /usr/local/DaticalDB

  1. Create a shared workspace outside of the user account

    $ sudo mkdir /usr/local/daticalws
    $ sudo chmod 777 /usr/local/daticalws/
    $ sudo vi /usr/localDaticalDB/datical.ini

    update the following lines, from

    -data
    @user.home/datical

    to

    -data
    /usr/local/daticalws
    -configuration
    /usr/local/DaticalDB/configuration
  2. Setup path to run datical and hammer

    $ sudo vi /etc/profile.d/datical_env.sh
    #!/bin/sh
    PATH="$PATH:/usr/local/DaticalDB:/usr/local/DaticalDB/repl"
  3. Start datical as root and complete the install

    $ sudo /usr/local/DaticalDB/datical
    1. Install JDBC Drivers
    2. Install a source control plugin if desired

  4. Set up the JDBC drivers
    • Create a datical.project file with the CLI driver location and copy to each project directory
