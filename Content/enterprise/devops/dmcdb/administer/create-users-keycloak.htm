Create users and groups in Keycloak

The User Settings page lists all users who can log into the (). This view is only available to users with the System Administrator role. The view allows you to view, search, and edit users that have been created in Keycloak or imported into Keycloak from your LDAP server.

How do I add a user to the user settings list?

To create new users in the :

  1. Navigate to Keycloak and login.
  2. In the Manage section of the left navigation, click Users.
  3. In the upper right corner, select Add user.
  4. Enter a username, email, first and last name.

You can also enable or disable a user, request email verification, or require specific user actions from this screen.

  1. Click Save.

How do I add a group to the user settings list?

To create new users in the :

  1. Navigate to Keycloak and login.
  2. In the Manage section of the left navigation, click Groups.
  3. In the upper right corner, select the Create group button.
  4. Enter the title of the group.

You can also enable or disable a user, request email verification, or require specific user actions from this screen.

  1. Click Save.
