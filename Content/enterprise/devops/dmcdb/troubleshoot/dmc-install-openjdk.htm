Install with only OpenJDK installed

Problem

If only OpenJDK is installed and you have trouble installing DaticalDB.exe

Trouble installing Datical-DB-win-x86_64… with only the OpenJDK on your system

If you get an error similar to "This application requires a Java Runtime Environment 1.7.0"

and you have verified that java is indeed installed.

How to verify that OpenJDK is installed and in your path

Open a command prompt and type

java -version

It should return something like

openjdk version "1.8.0_242"OpenJDK Runtime Environment (build 1.8.0_242-b08)
			Eclipse OpenJ9 VM (build openj9-0.18.1, JRE 1.8.0 Windows Server 2012 R2 amd64-64-Bit Compressed References 20200122_564
			(JIT enabled, AOT enabled)
			OpenJ9   - 51a5857d2
			OMR      - 7a1b0239a
		JCL      - 8cf8a30581 based on jdk8u242-b08)

Solution

  1. The immediate solution

Try running this from the command line in the same directory as the DaticalDB executable

java -jar path_to_DaticalDB_exe_file

For instance

java -jar DaticalDB-win-x86_64-6.15.6361.exe

OR

2. Make sure that the administrator installed OpenJDK with the registry keys update selected which is NOT a default

Reinstall or repair java with this option checked
