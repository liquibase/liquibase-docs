Pipeline operation report view

The pipeline report view contains a list of operations that have been performed on pipeline steps and links to reports for those operations.

Navigating to the pipeline operation report view

To navigate to the pipeline operation report view:

  1. Click on My Projects
  2. Select a project
  3. Click on Reports

What does this view contain?

The pipeline operation report view contains a list of all operations reports which occur-ed in your pipeline. From this view you can see:

  • Date and Time the operation was performed
  • Name of the step in which the operation occurred
  • Report type
  • Result of the operation

To view your report, click the download button in the right column.

Filtering

The pipeline operations report view also allows you to filter your reports at the top of the page by any of the listed categories:

  • Report Type
  • Result
  • Step Name
  • Start Date
  • End Date

What will my report contain?

When you view your report, you will see:

  • When the operation that produced the report was performed
  • Who initiated the operation
  • The database and schema information for the report
  • A table impact summary
  • A list of simulated or deployed changes and the outcomes of each
