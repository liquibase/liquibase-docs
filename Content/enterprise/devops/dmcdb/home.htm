Deployment Monitoring Console database (DMCDB)

Overview of the DMCDB

The Deployment Monitoring Console Database (DMCDB) records Datical operations and status information for presentation in the Deployment Monitoring Console (DMC). This is available for Liquibase Enterprise versions 7.0 or higher.

Datical operations log records in the DMCDB:

  • Status

  • Forecast

  • Deploy

Creating & Initializing the DMCDB Database

For documentation on the requirements for and provisioning of the DMCDB, see the installation documentation for the Deployment Monitoring Console (DMC).

Configuring Projects for DMCDB Use

The client would configure objects for the Audit Database (predecessor to the DMCDB) in releases of prior to 7.0. This has changed with the switch to the DMCDB in versions 7.0 and later. You should not configure projects to use a DMCDB until the DMC is fully installed and operational. If projects are configured to use the DMCDB prior to the complete installation of the DMC, the DMC installation may encounter issues.

For more information on configuring Datical projects to report to a DMCDB refer to the DMC documentation for Registering with the DMCDB
