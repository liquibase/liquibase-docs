Place Files in the SCM Repository

SQL scripts may be organized in two ways:

  • Fixed Folders: organized by type into specifically named folders. See Fixed Folder Names.
  • Flexible Folders: organized into folders that you name. Properties in the metadata.properties file determine how they are processed. See Flexible Folder Names.

See also overviews of packager workflows and guidelines for writing scripts for packager:

  • Overview of Packaging
  • Write SQL Scripts for [%=General.LBEnterprise%]

Liquibase changelogs are placed into a flexible folder with required properties in metadata.properties set. See Packaging Liquibase Changelogs.
