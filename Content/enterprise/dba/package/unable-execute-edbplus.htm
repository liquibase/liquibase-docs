Troubleshooting: Unable to execute "datical_edbplus.sh -v"

Problem

While setting up packager in my CI tool I'm getting this error.

10:08:47.906 [main] ERROR SqlTool - Error: Unable to execute "datical_edbplus.sh -v". Check to make sure datical_edbplus.sh is in your path. Check the log file for more details.

Solution

provides "edbplus" client as part of installation.

Make sure that you have "edbplus" folder in your Datical_install_folder as follows:

Next, add this folder to your PATH environment variable. Based on the screenshot above, the folder to include in PATH environment variable would be: C:\Users\Administrator\DaticalDB\edbplus
