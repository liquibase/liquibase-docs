Structured Logging Keys

uses a Mapped Diagnostic Context (MDC) to generate structured logs in a JSON format. The output is a nested group of key-value pairs. You can access data from your structured logs by calling on the keys described on this page.

Most values in the log are strings. Other JSON types such as arrays and integers are used for some nested keys.

In 4.23.2+, you can create a custom metadata file that controls exactly which keys appears in your s. For more information, see Custom and Private Data in [%=General.StructuredLog%]s.
