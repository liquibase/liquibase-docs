ConfigCat Feature Flags Extension

Precondition to control the execution of a changelog or changeset based on the state of the feature flag in ConfigCat.

Supported Editions

Installation

The easiest way to install this extension is with lpm liquibase package manager.

lpm update lpm add configcat

Setup

A SDK Key is required for the extension to access the ConfigCat API.

--config-cat-sdk-key=PARAM SDK Key for ConfigCat (liquibase.configCat.sdkKey) (LIQUIBASE_CONFIG_CAT_SDK_KEY) [deprecated: --configCatSdkKey]

Usage

To use this extension, add the configCatFeatureFlag precondition to your Changelog or Changeset with an enabledFlags attribute. The value for enabledFlags is either a string with one feature flag key or a comma separated string with multiple feature flag keys. All feature flags must be enabled for the precondition to pass.

Example

databaseChangeLog: - preConditions: - configCatFeatureFlag: enabledFlags: changelog-testing <changeSet id="1" author="example"> <preConditions> <ext:configCatFeatureFlag enabledFlags="changelog-testing"/> </preConditions> ... </changeSet>
