Node Package Manager (NPM)

The Node Package Manager allows you to integrate into your current application using the same programming language you are familiar with. The packager acts as a library and it enables you to install pre-built packages that help you maintain your database easily and efficiently. Developers can easily integrate it in the current application instead of using the separate Java CLI.

This installation method was developed with TypeScript and JavaScript engineers in mind. It provides a smooth user experience between the NPM packager and the CLI tool. This is made possible by transpiling both an ES Module and a Common JS module in the dist code. The Node CLI of this package can be used with node-liquibase. All 'top level' commands are implemented in this package. Here's a complete listing of commands that have been implemented.

Be sure to only import from the module name when importing the modules to use them in a JavaScript or TypeScript file. This package name is required due to dependent projects.
