Secrets Management

Secrets Management Extensions help protect credentials needed to access your database as well as any other properties you may consider sensitive. The extensions allow for properties to point to a secret stored in an external secrets vault by a key. During runtime, will fetch the value for this secret. Temporary known limitation: only one secrets manager extension can be installed at a time.

Features

  • Meet security requirements
  • Have public user documentation available
  • Configuration and dependencies not likely to change
  • Best effort to make future versions backwards compatible
