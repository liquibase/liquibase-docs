Ant dropAllDatabaseObjects

Drops all database objects owned by the user.

Functions, procedures and packages are not dropped (limitation in 1.8.1).

Syntax

<liquibase:dropAllDatabaseObjects>  
    <liquibase:database  driver="${db.driver}"  url="${db.url}"  user="${db.user}"  password="${db.password}"/>  
</liquibase:dropAllDatabaseObjects>

Parameters

Attribute Description Required
promptOnNonLocalDatabase No; default is false.
classpathref No
databaseref Yes, unless a nested <database> element is present.
driver No
url No
username No
password No
defaultSchemaName No
currentDateTimeFunction No
databaseChangeLogTableName No
databaseChangeLogLockTableName No
logLevel No

Nested s

Description Required
classpath Optional
database Required unless a databaseref is given.
changelogparameters Optional
