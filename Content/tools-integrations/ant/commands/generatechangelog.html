Ant generateChangeLog

Generates a to recreate an existing database.

Syntax

<liquibase:generateChangeLog  classpathref="classpath">  
    <liquibase:database  driver="${driver.class}"  url="${jdbc.url}"  user="${jdbc.user}"  password="${jdbc.user}"/>  
    <liquibase:xml  outputFile="/path/to/output/changelog.xml"  encoding="UTF-8"/>  
</liquibase:generateChangeLog>

Generates a file for the database in XML format.

<liquibase:generateChangeLog  classpathref="classpath">  
    <liquibase:database  driver="${driver.class}"  url="${jdbc.url}"  user="${jdbc.user}"  password="${jdbc.user}"/>  
    <liquibase:xml  outputFile="/path/to/output/changelog.xml"  encoding="UTF-8"/>  
    <liquibase:yaml  outputFile="/path/to/output/changelog.yaml"  encoding="UTF-8"/>  
</liquibase:generateChangeLog>

Generates the in both XML and YAML format.

Parameters

Attribute Description Required

promptOnNonLocalDatabase No; default is false.

driver No
url No

Nested s

XML, YAML, JSON, or TXT

This task is capable of generating files in multiple formats. At least one of these elements is required.

Attribute Description Required
outputFile Yes
encoding No. Defaults to system encoding

For example:

<liquibase:xml  outputFile="/path/to/output/changelog.xml"  encoding="UTF-8"/>
