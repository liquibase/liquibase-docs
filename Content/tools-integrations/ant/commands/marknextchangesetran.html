Ant markNextChangesetRan

Marks the next change as already ran. Useful for when a change was made manually and so an update is failing.

Syntax

<liquibase:markNextChangesetRan  changeLogFile="/path/to/changelog.xml">  
    <liquibase:database  driver="${db.driver}"  url="${jdbc.url}"  user="${db.user}"  password="${db.password}"/>  
</liquibase:markNextChangesetRan>

Parameters

Attribute Description Required
changeLogFile Yes
contexts No
outputFile No
outputEncoding No; defaults to system encoding.
promptOnNonLocalDatabase No; default is false.
classpathref No
databaseref Yes, unless a nested <database> element is present.
driver No
url No
username No
password No
defaultSchemaName No
currentDateTimeFunction No
databaseChangeLogTableName No
databaseChangeLogLockTableName No
logLevel No

Nested s

Description Required
classpath Optional
database Required unless a databaseref is given.
changelogparameters Optional
