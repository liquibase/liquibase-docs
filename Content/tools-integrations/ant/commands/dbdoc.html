Ant dbDoc

Generates db-doc database documentation for a given database.

Syntax

<liquibase:dbDoc  changeLogFile="/path/to/changeLog.xml"  outputDirectory="/path/to/output/dir">  
	<liquibase:database  driver="${db.driver}"  url="${db.url}"  user="${db.user}"  password="${db.pasword}"/>  
</liquibase:dbDoc>

A basic example of generating database documentation for a given file.

<liquibase:dbDoc  databaseRef="my-database"  changeLogFile="/path/to/changelog.xml"  outputDirectory="/path/to/output/dir"/>

Generates database documentation for the referenced database.

Parameters

Attribute Description Required
changelogFile Yes
contexts No
outputDirectory Yes
classpathref No
databaseref Yes, unless a nested <database> element is present.
promptOnNonLocalDatabase No; default is false.
driver No
url No
username No
password No
defaultSchemaName No
currentDateTimeFunction No
databaseChangeLogTableName No
databaseChangeLogLockTableName No
logLevel No

Nested s

Description Required
classpath Optional
database Required unless a databaseref is given.
changelogparameters Optional
