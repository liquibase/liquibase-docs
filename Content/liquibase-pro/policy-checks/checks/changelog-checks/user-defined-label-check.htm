UserDefinedLabelCheck

This check confirms that a specific label or a label that matches a specific pattern is present on all changesets.

About UserDefinedLabelCheck
Liquibase version required 4.25.0+
Scope (--checks-scope) changelog
Default status disabled
Default severity (exit code) 0 ("INFO")
Customizable settings Yes (dynamic)

Uses

Use the check to confirm when a specific label or a label that matches a specific pattern is present on all s. More specifically, UserDefinedLabelCheck uses a regular expression to ensure that the label fits a particular Jira or CRQ format.
