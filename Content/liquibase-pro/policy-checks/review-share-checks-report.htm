Review and Share Your Checks Report

Automatically generate a Checks Report using the Liquibase checks run command. The Checks Report gives you an easy-to-read, sharable summary of command metadata, warning messages, check severity levels, checks packages, and configuration details, plus an organized summary of check output in a browser page.

After a Policy Check runs successfully, the table output appears. The output filters according to the --checkname argument which includes a specific checkname, a list of checknames, and wildcard checknames. The only exception is in the case of checks delete --checkname=checktodelete which displays the full table.

    liquibase checks run --checkname=python*, liquibase checks run --checkname=*python, liquibase checks run --checkname=*python*

    liquibase checks run --checkname=mySpecificCheck

    liquibase checks run --checkname=mycheck1,mycheck2,mycheck3
