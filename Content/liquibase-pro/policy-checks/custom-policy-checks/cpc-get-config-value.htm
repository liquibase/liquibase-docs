Retrieve any configuration property in a Custom Policy Check

The liquibase_utilities.get_config_value python helper function allows you retrieve any configuration property which includes environment variables, system properties, flow files, flow variables, and checks run command line arguments.

Syntax: liquibase_utilities.get_config_value("CONFIGURATION_PROPERTY_VALUE")

Before you begin

  • [%=General.LBPro%] Custom Policy Checks

Step-by-step

  1. Navigate to your Custom Policy Check Python script and add the configuration value desired. This configuration value could be an environment variable, a system property, flow file, flow variable, or a checks run argument.
    Here is the syntax: config_value=liquibase_utilities.get_config_value("your-config-value")
  2. During the SCRIPT_ARGS step of Custom Policy Check creation, run the get_config_value command to pull the

    In this example, we will use the checks run changelogfile argument:
     config_value=liquibase_utilities.get_config_value("changelogfile")
  3. Navigate to the CLI and run JAVA_OPTS="TEST_SCRIPT_ARG="changelogfile"
    In the output you will see the script arg specified, which is changelogfile in this example.
