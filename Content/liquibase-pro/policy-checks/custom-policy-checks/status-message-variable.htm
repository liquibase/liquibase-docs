Status Message Variables for Custom Policy Checks

The status message appears at the conclusion of a custom policy check's execution. Your Python script can contain variables to provide information specific to the custom check. This guide shows where those are set up within a Python script and how can expand those variables into useful values.

The Checks extension, which includes all Custom Policy Check functionality, is available to Pro 4.29.0+ users.

Review this set of Python script examples and search for liquibase_status.message to see example set ups of status messages with replaceable strings.

Prerequisites

This guide picks up the Option for advanced users step on the Custom Policy Checks page. Refer to that page for setting up custom policy checks.
