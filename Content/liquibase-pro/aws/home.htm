Integrate AWS with

Introduced in 4.31.0

integrates the use of several Amazon features which include remote file access, secrets management, and flexible-schema NoSQL database management. You can easily incorporate into your AWS workflow with this feature.

The AWS extension encompasses what used to be the three AWS S3, AWS Secrets Manager, and Amazon DynamoDB Pro extensions into one. This single, convenient source provides a unified user experience and allows for seamless work between the three tools.

Features

Each element of this extension manages different aspects of AWS content. Find out more about each of their capabilities within in the list below.

  • AWS S3 - Store and retrieve data from remote file locations in a secure, centralized way.
  • AWS Secrets Manager - Seamlessly access sensitive authentication details from your AWS Secrets Manager.
  • Amazon DynamoDB Pro - Manage DynamoDB objects and data via PartiQL statements.

Limitations

  • Compatible with 4.31.0 and later
