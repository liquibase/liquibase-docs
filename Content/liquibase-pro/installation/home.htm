Install Liquibase Pro with Maven

This article provides an overview of how and why to install Liquibase Pro using Maven, including links to detailed, platform-specific instructions for Windows, Mac, and Linux/Unix systems.

If you’re already using Maven with Liquibase Open Source,refer to our guide on Upgrading from Liquibase Open Source to Liquibase Pro, available on the Getting Started with Liquibase and Maven page.

Some Maven goals are only available with Liquibase Pro. The Maven Goals page contains a list of Liquibase goals available for use with Maven and all Liquibase Pro specific goals are marked with a PRO label.
