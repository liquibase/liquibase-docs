AWS 4.31.0+ integrates the three AWS S3, AWS Secrets Manager, and Amazon DynamoDB Pro extensions into one. This single, convenient source provides a unified user experience and prevents conflicts between the three tools. Maven: Liquibase AWS Extension
AWS S3 integrates the use of Amazon S3 to store and retrieve data from anywhere. Extending to use remote file locations enables secure, centralized file management. Maven: S3 Remote Accessor Extension
AWS Secrets Manager You can natively read your application secrets in with the AWS Secrets Manager extension. This extension allows you to store sensitive authentication details such as your URL, username, password, and license key in your vault. Maven: Liquibase AWS Secrets Manager Extension
HashiCorp Vault You can natively read your application secrets in with the Hashicorp Vault extension. This extension allows you to store sensitive authentication details such as your URL, username, password, and license key in your vault. Maven: HashiCorp Vault Extension
Amazon DynamoDB Pro The DynamoDB Pro extension lets you manage DynamoDB objects and manage data via PartiQL statements. It handles AWS platform instructions asynchronously while supporting features including Policy Checks, Secrets Management, and [%=General.StructuredLogging%]. Maven: Liquibase DynamoDB Commercial Extension
MongoDB Pro The MongoDB Pro extension lets you use all the features of with MongoDB, including Policy Checks, Secrets Management, and [%=General.StructuredLogging%]. You can use with several MongoDB providers, including MongoDB Atlas, MongoDB Community Server, MongoDB Enterprise Server, and the Amazon DocumentDB clone. Maven: Liquibase MongoDB Commercial Extension
Policy Checks

In 4.31.0+, the Checks extension is included when you download . The Checks Extension contains all Policy Checks functionality for all supported database platforms. This includes many built-in checks to provide better risk mitigation, compliance, code quality, security, and governance to your and database. You can also write Custom Policy Checks in Python and integrate them into the policy checks framework for easy and consistent deployment.

 Maven: Liquibase Checks
