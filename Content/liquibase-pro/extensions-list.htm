Some features are stored in separate extensions, which are modular packages of code that may not follow the release cadence. The following is a list of such extensions:

Amazon AWS

AWS Liquibase Extension Documentation Description Download
Integrate AWS with [%=General.LBPro%] 4.31.0+ integrates the three AWS S3, AWS Secrets Manager, and Amazon DynamoDB Pro extensions into one. This single, convenient source provides a unified user experience and prevents conflicts between the three tools. Bundle: Maven: Liquibase AWS Extension

Azure

Azure Liquibase Extension Documentation Description Download
Integrate Azure with [%=General.LBPro%] In 4.32.0+, you can use the Liquibase Azure extension to access files from your azure storage account. Bundle: Maven: Liquibase Azure Extension

Sensitive credential storage

Liquibase Extension Documentation Description Download
HashiCorp Vault You can natively read your application secrets in Liquibase with the Hashicorp Vault extension. This extension allows you to store sensitive Liquibase authentication details such as your URL, username, password, and license key in your vault. Maven: HashiCorp Vault Extension

Databases

Liquibase Extension Documentation Description Download
Amazon DynamoDB Pro The Liquibase DynamoDB Pro extension lets you manage DynamoDB objects and manage data via PartiQL statements. It handles AWS platform instructions asynchronously while supporting features including Policy Checks, Secrets Management, and [%=General.StructuredLogging%]. Download the extension individually or get the AWS extension bundle. Individual: Maven: Liquibase DynamoDB Commercial Extension
MongoDB Pro The Liquibase MongoDB Pro extension lets you use all the features of with MongoDB, including Policy Checks, Secrets Management, and [%=General.StructuredLogging%]. You can use Liquibase with several MongoDB providers, including MongoDB Atlas, MongoDB Community Server, MongoDB Enterprise Server, and the Amazon DocumentDB clone. Maven: Liquibase MongoDB Commercial Extension

Remote file management

AWS Liquibase Extension Documentation Description Download
AWS S3 Liquibase integrates the use of Amazon S3 to store and retrieve data from anywhere. Extending Liquibase to use remote file locations enables secure, centralized file management. Individual: Maven: S3 Remote Accessor Extension

Policy Checks

Liquibase Extension Documentation Description Download
Policy Checks

In Liquibase 4.31.0+, the Checks extension is included when you download Liquibase. The Liquibase Checks Extension contains all Policy Checks functionality for all supported database platforms. This includes many built-in checks to provide better risk mitigation, compliance, code quality, security, and governance to your and database. You can also write Custom Policy Checks in Python and integrate them into the Liquibase policy checks framework for easy and consistent deployment.

 Maven: Liquibase Checks

Deprecated Liquibase Pro extensions

In Liquibase Pro 4.30.0 and earlier, you can download these extensions individually.
These are deprecated and we recommend upgrading to Liquibase Pro 4.31.1 or later to get the AWS extension bundle that contains all three.
