As database projects become larger and more complex, it becomes critical to scale a database management tool to support more environments, larger teams, and changing compliance and security requirements.

extends the capabilities of to support the agile and iterative development necessary to apply DevOps practices and patterns to database workflows.

Capabilities

offers advanced capabilities including database and changelog policy checks, automated drift detection, and more. You can activate these capabilities by applying your license key.

To learn more about the capabilities of , visit the following documentation pages:

  • [%=General.Liquibase%] Flow Files: portable, platform-independent Liquibase workflows. Flow files group multiple commands to run sequentially using a single command.
  • Native Executors: used to run highly specialized SQL when the JDBC driver is insufficient.
  • [%=General.Liquibase%] Observability:
    • Operation Reports: portable reports for operational insights and error detection.
    • [%=General.StructuredLogging%]: groups relevant pieces of information in JSON formatted messages which can be easily sent to dashboards to be parsed into actionable insights.
  • Policy Checks: used to check the quality and security of the database code on-demand or in automation against a set of rules.
  • Remote Files: use Amazon S3 to store and retrieve data from anywhere, enabling secure, centralized file management for Liquibase operations.
  • Secrets Management: securely store your database credentials and other secrets for Liquibase with AWS Secrets Manager and HashiCorp Vault.
  • Stored Logic: captures stored logic objects including procedures, functions, packages, and triggers.
  • Targeted Rollbacks: commands that provide precision changeset rollback without impacting surrounding changesets.

Many features come pre-installed with Liquibase. Some other features can be downloaded in separate extensions. For a list of such extensions, see [%=General.LBPro%] Extensions.
